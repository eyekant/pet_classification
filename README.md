# Pet Breed Classification
Designed a CNN from scratch and fine-tuned a ResNet-18 model with Pytorch to classify cats and dogs images according to their breed.

## Available files
- `assignment_module_two.ipynb`: main file
- `source`
  - `results_seed42.json`: results of training (with the ablated models) 
- `ckpts`
  - all available model weights
